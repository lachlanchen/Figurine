# Rara-chan Panda Doll Short Movie

This folder contains the creative plan and generation prompts for a short social video made with Jianying / XYQ image-to-video tools.

## Core Assets

Use these as visual references when generating clips:

- Logo: `../website/assets/brand/lazyingart-logo.png`
- Panda figure: `../website/assets/products/lala-chan-panda-doll/lala-chan-panda-doll-product-v1.webp`
- Solo lifestyle reference: `../website/assets/products/lala-chan-panda-doll/lala-chan-panda-doll-lifestyle-alone-v1.webp`
- Human-scale reference: `../website/assets/products/lala-chan-panda-doll/lala-chan-panda-doll-lifestyle-friends-v1.webp`

## Concept

Title: **Rara-chan's Tiny Luck Delivery**

A tiny handmade panda doll wakes up on a clean studio table, discovers the LazyingArt mark, and performs a small charming mission: delivering a little spark of luck to the viewer. The tone is bright, cute, premium, and slightly magical without becoming childish.

Recommended format:

- Length: `15-20 seconds`
- Aspect ratio: `9:16` for social posts, `16:9` for website hero reuse
- Style: premium toy commercial, soft daylight, playful micro-story
- Music: light xylophone, soft bass, tiny bell hits
- CTA: `Rara-chan is available now.`

## Files

- `short-film-script.md` - story, timing, voiceover, captions, and sound notes.
- `jianying-prompts.md` - copy-ready prompts for image-to-video generation.
- `shot-list.md` - compact production checklist and export notes.

## Usage

1. Open `https://xyq.jianying.com/home?tab_name=home`.
2. Choose an image-to-video or video generation workflow.
3. Upload the panda figure as the primary character reference.
4. Generate each shot from `jianying-prompts.md`.
5. Assemble the clips in the order from `short-film-script.md`.
6. Add the LazyingArt logo only in the final end card.

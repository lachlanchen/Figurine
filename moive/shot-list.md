## Recommended Generation Settings

- Primary version: `9:16`, `1080x1920`, `18s`
- Website version: `16:9`, `1920x1080`, `18s`
- Clip generation: create each shot separately, then assemble.
- Camera: slow push, slow drift, gentle orbit. Avoid fast cuts.
- Motion: tiny plush movement, not full cartoon character animation.
- Color: warm cream, soft white, black panda details, tiny gold light accent.

## Editing Notes

- Use the actual logo file for the end card: `../website/assets/brand/lazyingart-logo.png`.

- Add captions as editor text layers so spelling stays correct.
- Use `buy.lazying.art` only on the end card.

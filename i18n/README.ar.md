# Figurine

واجهة متجر ثابتة متعددة اللغات من LazyingArt لدفاتر الجلد المرقع، دمى الباندا Rara-chan / Lala-chan، قلادات Lucky Paw الجلدية، وملحقات AI الاختيارية.

## المنتجات

| المنتج | الدفع |
| --- | --- |
| دفتر جلد مرقع | A4 `USD 198`، B5 `USD 168`، A5 `USD 148`، A6 `USD 118` |
| دمية الباندا Rara-chan / Lala-chan | `USD 198` |
| قلادة Lucky Paw الجلدية | `USD 98`، مع اختيار اللون الداخلي والخارجي |
| ملحقات AI | إضافة `USD 300` للتسجيل والدردشة وملاحظات AI |

يمنح الكود `LAZY` خصم `12%` في Stripe Checkout. مدة تنفيذ القطع اليدوية حوالي `1` إلى `1.5` شهر.

## الموقع و i18n

الموقع الثابت موجود في [website/](../website/) وينشر إلى `buy.lazying.art` عبر GitHub Pages. مفاتيح الترجمة ومنطق التفاف النص الذكي موجودة في [website/script.js](../website/script.js).

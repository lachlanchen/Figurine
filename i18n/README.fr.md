## Produits

| Produit | Paiement |
| --- | --- |
| Carnet en cuir patchwork | A4 `USD 198`, B5 `USD 168`, A5 `USD 148`, A6 `USD 118` |
| Poupée panda Rara-chan / Lala-chan | `USD 198` |
| Pendentif cuir Lucky Paw | `USD 98`, couleurs intérieure et extérieure |

Le code `LAZY` donne `12%` de remise dans Stripe Checkout. Le délai artisanal est d'environ `1` à `1.5` mois.

## Site et i18n

Le site statique se trouve dans [website/](../website/) et est publié sur `buy.lazying.art` via GitHub Pages. Les traductions et la logique de retour à la ligne intelligente sont dans [website/script.js](../website/script.js).

## Stripe

Les Stripe Payment Links sont générés depuis `../Stripe`. Les secrets restent dans `../Stripe/.env`; la configuration publique est exportée vers [website/stripe-config.js](../website/stripe-config.js).

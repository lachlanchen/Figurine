# Figurine

Tienda estática multilingüe de LazyingArt para cuadernos de cuero patchwork, muñecos panda Rara-chan / Lala-chan, colgantes de cuero Lucky Paw y accesorios de IA opcionales.

## Productos

| Producto | Pago |
| --- | --- |
| Cuaderno de cuero patchwork | A4 `USD 198`, B5 `USD 168`, A5 `USD 148`, A6 `USD 118` |
| Muñeco panda Rara-chan / Lala-chan | `USD 198` |
| Colgante de cuero Lucky Paw | `USD 98`, colores interior y exterior |
| Accesorios IA | `USD 300` extra para grabación, chat y notas IA |

El código `LAZY` aplica `12%` de descuento en Stripe Checkout. El plazo artesanal es de `1` a `1.5` meses.

## Web e i18n

El sitio estático está en [website/](../website/) y se publica en `buy.lazying.art` con GitHub Pages. Las traducciones y el ajuste inteligente de líneas viven en [website/script.js](../website/script.js).

## Stripe

Los Stripe Payment Links se generan desde `../Stripe`. Los secretos quedan en `../Stripe/.env`; la configuración pública se exporta a [website/stripe-config.js](../website/stripe-config.js).

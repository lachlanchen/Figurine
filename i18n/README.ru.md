# Figurine

Многоязычный статический магазин LazyingArt для блокнотов из лоскутной кожи, кукол-панд Rara-chan / Lala-chan, кожаных подвесов Lucky Paw и дополнительных AI-аксессуаров.

## Товары

| Товар | Оплата |
| --- | --- |
| Блокнот из лоскутной кожи | A4 `USD 198`, B5 `USD 168`, A5 `USD 148`, A6 `USD 118` |
| Кукла-панда Rara-chan / Lala-chan | `USD 198` |
| Кожаный подвес Lucky Paw | `USD 98`, выбор внутреннего и внешнего цвета |
| AI-аксессуары | Дополнительно `USD 300` для записи, чата и AI-заметок |

Код `LAZY` дает скидку `12%` в Stripe Checkout. Срок ручного изготовления около `1`-`1.5` месяца.

## Сайт и i18n

Статический сайт находится в [website/](../website/) и публикуется на `buy.lazying.art` через GitHub Pages. Переводы и умный перенос строк находятся в [website/script.js](../website/script.js).

## Stripe

Stripe Payment Links создаются из `../Stripe`. Секреты хранятся в `../Stripe/.env`; публичная конфигурация экспортируется в [website/stripe-config.js](../website/stripe-config.js).

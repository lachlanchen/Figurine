# Figurine

LazyingArt의 다국어 정적 스토어입니다. 패치워크 가죽 노트, Rara-chan / Lala-chan 판다 인형, Lucky Paw 가죽 펜던트, 선택형 AI 액세서리를 판매합니다.

## 제품

| 제품 | 결제 |
| --- | --- |
| 패치워크 가죽 노트 | A4 `USD 198`, B5 `USD 168`, A5 `USD 148`, A6 `USD 118` |
| Rara-chan / Lala-chan 판다 인형 | `USD 198` |
| Lucky Paw 가죽 펜던트 | `USD 98`, 안쪽/바깥쪽 색상 선택 |
| AI 액세서리 | 녹음, 채팅, AI 노트를 위해 추가 `USD 300` |

Stripe 결제에서 프로모션 코드 `LAZY`를 사용하면 `12%` 할인됩니다. 수작업 제품 리드타임은 약 `1`에서 `1.5`개월입니다.

## 웹사이트와 i18n

정적 사이트는 [website/](../website/)에 있으며 GitHub Pages로 `buy.lazying.art`에 배포됩니다. 번역 키와 스마트 줄바꿈은 [website/script.js](../website/script.js)에 있습니다.

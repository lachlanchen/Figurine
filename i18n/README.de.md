Mehrsprachiger statischer LazyingArt-Shop für Patchwork-Ledernotizbücher, Rara-chan / Lala-chan Panda-Puppen, Lucky Paw Lederanhänger und optionale AI-Zubehörteile.

## Produkte

| Produkt | Checkout |
| --- | --- |
| Patchwork-Ledernotizbuch | A4 `USD 198`, B5 `USD 168`, A5 `USD 148`, A6 `USD 118` |
| Rara-chan / Lala-chan Panda-Puppe | `USD 198` |
| Lucky Paw Lederanhänger | `USD 98`, Innen- und Außenfarbe wählbar |
| AI-Zubehör | Zusätzlich `USD 300` für Aufnahme, Chat und AI-Notizen |

Der Code `LAZY` gibt `12%` Rabatt in Stripe Checkout. Die handgemachte Lieferzeit beträgt etwa `1` bis `1.5` Monate.

## Website und i18n

Die statische Website liegt in [website/](../website/) und wird mit GitHub Pages auf `buy.lazying.art` bereitgestellt. Übersetzungen und Smart-Wrapping stehen in [website/script.js](../website/script.js).

## Stripe

Stripe Payment Links werden aus `../Stripe` generiert. Secrets bleiben in `../Stripe/.env`; die öffentliche Konfiguration wird nach [website/stripe-config.js](../website/stripe-config.js) exportiert.
